zebra TC73 Mobile Device-Android version 14

Android App:

	Scanning Picklist through camera.
	If the picklist exist in DB the picklists will show as green tick in camera.
	User tap the green tick list.
	After pickup the picklists in DB status will updated as picked.

Website:

	UI for user uploading picklists in excel to DB.

Database:

	PostgreSQL(currently using) and MySQL both code available.

Zebra SDK:
	
	To decode Barcodes,Found Value and for Camera usage zebra sdk used.
	They are,
		AI-Data-Capture-SDK-3.2.3.aar
		barcode-localizer-5.0.2.aar

SDK Download Link:

	https://techdocs.zebra.com/ai-datacapture/latest/model/barcode-localizer/
	https://zebratech.jfrog.io/ui/repos/tree/General/emc-mvn-ext/com/zebra/ai/models/barcode-localizer/
